# 1 系统通用接口

---
## 1.1 获取验证码

* url : /common/get_smscode
* 接口编号 : 11
* 请求类型 : post

### 请求参数

|字段|类型|可否为空|描述|
|:--:|:--:|:--:|:--:|:--:|
|phone|string|否|手机号|
|cmd|int|否|验证码功能|


### cmd

|字段|值|描述|
|:--:|:--:|:--:|
|cmd|1|登录|
|cmd|2|修改密码|


### 返回数据

```
{
  code:'1000'
  msg:'suc'
  data: 'suc'
}
```


### 错误码

|code|meaning|
|:--:|:--|
|1101|超过验证码限制|
