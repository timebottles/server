# 4 时光瓶模块

--------------------------------------------------------------------   
## 4.1 创建时光瓶

### 接口概述

* url : /bottle/create
* 接口编号 : 41
* 请求类型 : post


### 请求参数


|字段|类型|可否为空|描述|
|:--|:--|:--|:--|:--|
|name|string|非空|瓶子名称|
|uid|string|非空|创建者id|
|des|string|可空|瓶子描述|
|type|int|可空|瓶子类型【0：分开，1：私密】|


### 返回数据

```
{
  code:'1000'
  msg:'suc'
  data: {
    'bid':'6666'
  }
}
```

### 错误码

|code|meaning|
|:--:|:--|
|4101|创建失败|
